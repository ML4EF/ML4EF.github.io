Course Details

This is an applied course in Machine Learning intended for students of Economics and Finance.

Programming Language:

The official language of this course is Python (3.5). We will use Anaconda in class.

You are expected to know how to program in Python before taking this class.

Grading:

Each class will be structured as follows:
- 30 minute quiz
- 15 minute break
- 1 hour hands-on lesson
- 15 minute break
- 1 hour test

Grades will be based on quizzes, tests and in-class performance.

Materials

The following materials are useful for the course:
- The Elements of Statistical Learning: Data Mining, Inference and Prediction
- Applied Predictive Modeling
- Probability Theory Review
- Linear Algebra Review
- Excellent lecture notes on Statistical Learning Theory
- Scikit-Learn
